    ```sh
  cd FinSight
	```
## Run BackEnd
	```sh
	cd  backend
	```
### Run Backend Nodejs
	```sh
	cd  FinSight
	```
- install package :
	```sh
	npm install
	```
- run nodejs :
  	```sh
	npm start
	```

### RunBackend Python
	```sh
	cd  FinSightAI
	```
#### Tạo môi trường env
- **Dùng Conda** 
  - Tạo môi trường :
  ```sh
  conda create -n env python==3.9.16
  ```
  - Activate môi trường :
  ```sh
  conda activate env
  ```
  - Install thư viện :
  ```sh
  pip install -r requirements.txt
  ```
- **Dùng venv** 
  - Tạo môi trường :
  ```sh
  python3 -m venv env
  ```
  - Activate môi trường :
  
  *Window*
  ```sh
  env\Scripts\activate.bat
  ```
  *Linux*
  ```sh
  source env/bin/activate
  ```
  - Install thư viện :
  ```sh
  pip install -r requirements.txt
  ```
#### Run app
```sh
python app.y
```
